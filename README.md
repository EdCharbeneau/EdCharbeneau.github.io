# edcharbeneau.github.io

Uncompiled [Jekyll](//jekyllrb.com) source code for [edcharbeneau.com](//edcharbeneau.com).

### Steps to use

* Clone repo
* Install Jekyll
* Run `jekyll serve` in the project directory on the command line
